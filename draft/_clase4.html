Uso de CDN, tarjetas, carrusel, botones y sistema de grid para construir interfaces responsivas

1) Introducción

En la clase anterior se estudió qué es Bootstrap y por qué ayuda a construir interfaces web de forma rápida. En esta clase se trabajará de manera más práctica: se verá cómo incluir Bootstrap en una página y cómo usar algunos de sus componentes más comunes.

El objetivo no es memorizar todas las clases, sino comprender la lógica: Bootstrap funciona agregando clases directamente a las etiquetas HTML. Esas clases ya tienen estilos preparados para colores, espaciados, columnas, tarjetas, botones, carruseles y muchos otros elementos.

CDN

Permite usar Bootstrap sin descargarlo. Solo se agregan enlaces al CSS y al JavaScript desde internet.

Componentes

Son bloques ya diseñados, como tarjetas, botones, carruseles, alertas, menús y formularios.

Grid

Permite ordenar el contenido en filas y columnas para que la página se adapte a diferentes pantallas.

Idea clave: Bootstrap se usa escribiendo HTML normal, pero agregando clases predefinidas. Por ejemplo: btn btn-primary, card, container, row y col-md-6.

2) Cómo incluir Bootstrap por CDN

La forma más rápida de iniciar con Bootstrap es usar un CDN. Un CDN es un servicio que entrega archivos desde internet, por lo que no se necesita descargar Bootstrap manualmente al proyecto.

Plantilla mínima con Bootstrap

<!doctype html> <html lang="es"> <head> <meta charset="utf-8"> <meta name="viewport" content="width=device-width, initial-scale=1"> <title>Mi página con Bootstrap</title> <!-- CSS de Bootstrap --> <link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.8/dist/css/bootstrap.min.css" rel="stylesheet" integrity="sha384-sRIl4kxILFvY47J16cr9ZwB07vP4J8+LH7qKQnuqkuIAvNWLzeN8tE5YBujZqJLB" crossorigin="anonymous"> </head> <body> <div class="container mt-5"> <h1 class="text-primary">Hola Bootstrap</h1> <p class="lead">Esta página ya puede usar las clases de Bootstrap.</p> <button class="btn btn-success">Comenzar</button> </div> <!-- JavaScript de Bootstrap. Debe ir antes de cerrar body. --> <script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.8/dist/js/bootstrap.bundle.min.js" integrity="sha384-FKyoEForCGlyvwx9Hj09JcYn3nv7wiPVlz7YYwJrWVcXK/BmnVDxM+D2scQbITxI" crossorigin="anonymous"></script> </body> </html>

Ejemplo real renderizado con Bootstrap por CDN

Hola Bootstrap

Esta página ya puede usar clases como container, text-primary, lead y btn.

Paso a paso:
  1. Crear un archivo llamado index.html.
  2. Agregar la etiqueta meta viewport para que la página sea responsiva.
  3. Colocar el enlace CSS de Bootstrap dentro de <head>.
  4. Escribir el contenido HTML dentro de <body>.
  5. Colocar el script de Bootstrap antes de </body>.
El archivo CSS permite usar estilos visuales. El archivo JavaScript permite que funcionen componentes interactivos como carruseles, dropdowns, modales y menús colapsables.

3) Tarjetas en Bootstrap

Las tarjetas o cards sirven para presentar contenido dentro de un bloque visual ordenado. Se usan mucho para mostrar cursos, productos, noticias, servicios, perfiles o recursos.

3.1 Tarjeta solo con texto

En este ejemplo, card crea el contenedor general, card-body agrupa el contenido, card-title define el título y card-text da formato al párrafo principal.

5) Botones de diferentes colores

Bootstrap ofrece botones con colores semánticos. Esto significa que el color comunica una intención: acción principal, éxito, advertencia, peligro, información, etc.

Ejemplo de botones

<button class="btn btn-primary">Primary</button> <button class="btn btn-secondary">Secondary</button> <button class="btn btn-success">Success</button> <button class="btn btn-danger">Danger</button> <button class="btn btn-warning">Warning</button> <button class="btn btn-info">Info</button> <button class="btn btn-light">Light</button> <button class="btn btn-dark">Dark</button>

Ejemplo real: colores de botones en Bootstrap

Clase Color / intención Ejemplo de uso
btn-primary Azul / acción principal Guardar, continuar, enviar formulario.
btn-secondary Gris / acción secundaria Cancelar, volver, opción menos importante.
btn-success Verde / resultado positivo Confirmar, aceptar, completar.
btn-danger Rojo / acción destructiva o crítica Eliminar, rechazar, advertir riesgo.
btn-warning Amarillo / advertencia Revisar, atención, acción con cuidado.
btn-info Celeste / información Ver detalles, aprender más, ayuda.
btn-light Claro / bajo contraste Acción suave sobre fondos oscuros.
btn-dark Oscuro / énfasis fuerte Acción destacada en diseños claros.

Botones con borde

<button class="btn btn-outline-primary">Primary</button> <button class="btn btn-outline-success">Success</button> <button class="btn btn-outline-danger">Danger</button> <button class="btn btn-outline-warning">Warning</button>

Las clases btn-outline-* crean botones con borde y fondo transparente. Son útiles cuando se desea una interfaz menos cargada visualmente.

6) Ejemplos del grid de Bootstrap

El grid de Bootstrap usa un sistema de 12 columnas. Con este sistema se puede decidir cuánto espacio ocupa cada elemento según el tamaño de pantalla.

6.1 Tres columnas iguales

<div class="container"> <div class="row"> <div class="col">Columna 1</div> <div class="col">Columna 2</div> <div class="col">Columna 3</div> </div> </div>

Cuando se usa solamente col, Bootstrap reparte el espacio disponible de forma automática.

6.2 Dos columnas de 6 espacios cada una

<div class="container"> <div class="row"> <div class="col-md-6">Contenido izquierdo</div> <div class="col-md-6">Contenido derecho</div> </div> </div>
Interpretación: col-md-6 significa que, desde pantallas medianas en adelante, cada bloque ocupa 6 de las 12 columnas. En pantallas pequeñas, los bloques se apilan uno debajo del otro.

6.3 Layout con contenido principal y barra lateral

<div class="container"> <div class="row g-4"> <main class="col-12 col-lg-8"> <h2>Contenido principal</h2> <p>Aquí se puede colocar el texto principal, artículos o productos destacados.</p> </main> <aside class="col-12 col-lg-4"> <h3>Barra lateral</h3> <p>Aquí pueden ir enlaces, noticias, filtros o información adicional.</p> </aside> </div> </div>

Este patrón es común en blogs, sitios de noticias, páginas de cursos y tiendas en línea.

6.4 Columnas responsivas para tarjetas

<div class="container"> <div class="row g-3"> <div class="col-12 col-sm-6 col-lg-3">Tarjeta 1</div> <div class="col-12 col-sm-6 col-lg-3">Tarjeta 2</div> <div class="col-12 col-sm-6 col-lg-3">Tarjeta 3</div> <div class="col-12 col-sm-6 col-lg-3">Tarjeta 4</div> </div> </div>

9) Cierre

Bootstrap permite construir páginas más rápido porque ofrece clases y componentes ya preparados. En esta clase se aplicaron ejemplos concretos: inclusión por CDN, tarjetas con texto e imagen, carrusel, botones de diferentes colores y diseños con grid.

Lo más importante es comprender qué clase se usa y por qué. Copiar ejemplos puede ayudar al inicio, pero el verdadero aprendizaje ocurre cuando se modifican, se prueban en diferentes pantallas y se adaptan a un proyecto propio.

Para recordar: una buena página con Bootstrap no depende solo de usar clases listas, sino de organizar correctamente el contenido con HTML semántico, elegir bien los componentes y probar el resultado en distintos tamaños de pantalla.
